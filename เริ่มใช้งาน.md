## 📊 ระบบวิเคราะห์แบบใหม่

### 1. RSI (Relative Strength Index)
บอกว่าราคาแพงหรือถูกเกินไป

- **RSI < 30** = ราคาถูกเกินไป → ควรซื้อ 💚
- **RSI 30-70** = ราคาปกติ → รอดูก่อน 🟡
- **RSI > 70** = ราคาแพงเกินไป → ควรขาย 🔴

### 2. Momentum (โมเมนตัม)
บอกว่าราคากำลังขึ้นหรือลงแรงแค่ไหน

- **> +2%** = กำลังขึ้นแรง → ซื้อเลย 🚀
- **0% ถึง +2%** = ขึ้นช้าๆ → ซื้อระวังๆ 📈
- **-2% ถึง 0%** = ลงช้าๆ → ขายระวังๆ 📉
- **< -2%** = กำลังลงแรง → ขายเลย 💥

### 3. Trend (แนวโน้ม)
บอกว่าราคากำลังไปทางไหน

- **UP** = กำลังขึ้น → เหมาะซื้อ ⬆️
- **DOWN** = กำลังลง → เหมาะขาย ⬇️
- **SIDEWAYS** = ไปมา → รอดูก่อน ↔️

### 4. Volatility (ความผันผวน)
บอกว่าราคาเปลี่ยนแปลงรุนแรงแค่ไหน

- **< 30%** = ผันผวนน้อย → เทรดได้มาก 😊
- **30-60%** = ผันผวนปานกลาง → เทรดปกติ 😐
- **> 60%** = ผันผวนมาก → เทรดน้อยๆ 😰

## 🛡️ ระบบป้องกันความเสี่ยง

### Stop Loss (ตัดขาดทุน)
- ถ้าขาดทุน **3%** จะขายอัตโนมัติ
- ตัวอย่าง: ซื้อที่ 2500 → ขายที่ 2425 (ขาดทุน 75 บาท)

### Take Profit (เก็บกำไร)
- ถ้ากำไร **5%** จะขายอัตโนมัติ
- ตัวอย่าง: ซื้อที่ 2500 → ขายที่ 2625 (กำไร 125 บาท)

### Position Sizing (ขนาดการเทรด)
ระบบคำนวณว่าควรเทรดเท่าไหร่:

- **มั่นใจมาก + ความเสี่ยงต่ำ** → เทรด 30-40% ของเงิน
- **มั่นใจปานกลาง** → เทรด 15-20% ของเงิน
- **มั่นใจน้อย + ความเสี่ยงสูง** → เทรด 5-10% ของเงิน

### Max Drawdown (ขาดทุนสูงสุด)
- ถ้าขาดทุนเกิน **20%** จะหยุดเทรดชั่วคราว
- ตัวอย่าง: เริ่มต้น 1500 บาท → ถ้าเหลือ 1200 บาท จะหยุดเทรด

## ⚙️ ปรับแต่งระบบ

### ถ้าอยากระวังมากขึ้น (Conservative)
แก้ไขใน `services/risk_manager.py`:
```python
self.stop_loss_pct = 2.0  # ตัดขาดทุนที่ -2% (เร็วขึ้น)
self.take_profit_pct = 3.0  # เก็บกำไรที่ +3% (เร็วขึ้น)
self.max_position_size_pct = 30.0  # เทรดสูงสุด 30%
```

แก้ไขใน `intelligent_trader.py`:
```python
self.min_confidence_to_trade = 70.0  # ต้องมั่นใจ 70% ขึ้นไป
```

### ถ้าอยากเสี่ยงมากขึ้น (Aggressive)
แก้ไขใน `services/risk_manager.py`:
```python
self.stop_loss_pct = 5.0  # ตัดขาดทุนที่ -5% (ช้าลง)
self.take_profit_pct = 8.0  # เก็บกำไรที่ +8% (ช้าลง)
self.max_position_size_pct = 50.0  # เทรดสูงสุด 50%
```

แก้ไขใน `intelligent_trader.py`:
```python
self.min_confidence_to_trade = 50.0  # มั่นใจ 50% ก็เทรดได้
```

## 🐛 แก้ปัญหา

### ปัญหา: ไม่มีการเทรดเลย
**สาเหตุ**:
1. ความมั่นใจต่ำเกินไป → ลดเกณฑ์ใน `intelligent_trader.py`
2. ความเสี่ยงสูงเกินไป → ตรวจสอบ volatility
3. ไม่มีสัญญาณชัด → รอตลาดเคลื่อนไหว

### ปัญหา: เทรดบ่อยเกินไป
**แก้ไข**: เพิ่มเกณฑ์ความมั่นใจ
```python
self.min_confidence_to_trade = 70.0  # เพิ่มจาก 60.0
```

### ปัญหา: ขาดทุนเรื่อยๆ
**ตรวจสอบ**:
1. Stop loss ทำงานหรือไม่? → ควรตัดที่ -3%
2. Risk management เปิดหรือไม่? → ดูใน log
3. ตลาดผันผวนมากหรือไม่? → ลดการเทรด
